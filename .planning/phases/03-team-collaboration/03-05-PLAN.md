In `src/repositories/memory-repository.ts`, replace the hardcoded `commented_memories: 0` in `countTeamActivity()` (line 466) with a third parallel SQL query.

The query counts distinct memory IDs from the comments table where:

- The comment's parent memory belongs to the given project (join memories on comments.memory_id = memories.id WHERE memories.project_id = projectId)
- The parent memory is not archived (memories.archived_at IS NULL)
- The comment was created after the `since` date (comments.created_at > since)

Add this as a third element in the existing `Promise.all([...])` array (lines 439-461).

The new query (third element in Promise.all):

```typescript
this.db
  .select({ count: sql<number>`count(distinct ${comments.memory_id})::int` })
  .from(comments)
  .innerJoin(memories, eq(comments.memory_id, memories.id))
  .where(
    and(
      eq(memories.project_id, projectId),
      isNull(memories.archived_at),
      gt(comments.created_at, since),
    )
  ),
```
